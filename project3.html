Project 3: Creative Animations

Creative Animations is an interactive collection of p5.js sketches that blend engineering concepts with artistic expression. Through generative art, physics simulations, and real-time user interactions, this project demonstrates my passion for combining code, creativity, and problem-solving.

From swirling particle systems that emulate fluid flow to dynamic shape morphing controlled by user input, each sketch aims to showcase both the imaginative and technical sides of programming.

  • Technologies: JavaScript, p5.js, HTML/CSS
  • Focus: Animations, Interactive Graphics, Creative Coding
